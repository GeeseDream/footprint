C语言
---------

在过去的工作中使用最多的是Java，同时也能熟练使用Python等，
然而没能完全掌握C语言。

现在，我将参考的教材《CPrimer Plus（第6版）》从 ``Hello World`` 开始继续学习有关C语言的一切。

    #include<stdio.h>

    int main()
    {
        printf("hello world\n");
        return 0;
